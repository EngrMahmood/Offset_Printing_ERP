Offset ERP - Full Easy Guide

Generated on: {{ generated_on|date:'Y-m-d H:i' }}

This guide explains every major ERP section in simple language for planner, production, dispatch, manager, and admin users. If you are new, read section by section in order.

0. Quick Access For Wrong Dropdown Values

  • If a wrong dropdown value is added (machine/operator/material/department), open Master Corrections.
  • Use Rename to correct spelling/name.
  • Use Deactivate for machine/operator when you want to stop future use.
  • Old records remain safe for audit.

1. Job Card Entry (Planning Stage)

  • Job Card No: unique order number for tracking.
  • SKU: product code.
  • PO Number/Date: customer order reference.
  • Material: paper or board used.
  • Colours: print colors (for example 4 or 1+1).
  • Order Qty: customer required finished pieces.
  • UPS: pieces per sheet.
  • Wastage: planned setup/loss sheets.
  • Machine: planned machine for this job.
  • Production Tolerance %: allowed extra sheets over planned quantity.

2. Job Card Auto Calculations

  • Required Sheets = Order Qty / UPS
  • Total Planned Sheets = Required Sheets + Wastage
  • Tolerance Sheets = Planned Sheets × Production Tolerance %
  • Allowed Sheets = Planned Sheets + Tolerance Sheets

3. Planned Time Calculations (Auto)

  • Estimated Run Time (min) = (Total Impressions Required / Machine Speed per hour) × 60
  • Estimated Setup Time (min) = Total Colors × Machine Setup Minutes per Color
  • Estimated Total Planned Time = Run Time + Setup Time

4. Production Entry (Execution Stage)

  • Job Card: choose which job is running.
  • Machine: auto-picked from job card, but override can be used when needed.
  • Operator, Shift, Date: who worked and when.
  • Impressions: total machine impressions done.
  • Output Sheets: good sheets produced.
  • Waste Sheets + Waste Reason: reject sheets and reason.
  • Run, Setup, Downtime minutes: actual consumed time.
  • Downtime Breakdown: add multiple downtime categories with minutes for one production entry.

5. Repeated Production Entries (Same Job Card)

  • Planned minutes are auto-allocated from remaining planned time.
  • Remaining Planned Time = Job Planned Total - Sum(previous active production planned times)
  • Total impressions are cumulative across production entries for the same job card.
  • Total impressions are validated against Total Impressions Required plus Production Tolerance %.
  • If extra allocation is needed beyond remaining planned time, overrun reason is mandatory.

6. Dispatch Entry (Delivery Stage)

  • Job Card: select the job being delivered.
  • Dispatch Date and Dispatch Qty: what was delivered and when.
  • DC No: optional dispatch challan reference.
  • Job completion threshold is 95% dispatch ratio.

7. OEE Rules

  • Availability uses unplanned downtime categories only: breakdown, operator, other (summed across all downtime rows).
  • Setup and planned/external losses are tracked, but treated differently from unplanned losses.
  • Performance compares actual impressions against machine expected impressions from run time.
  • Quality compares good sheets against quality-impact total sheets.

8. Overrun, Short Close, and Wastage Decisions

  • Overrun Minutes = (Run + Setup + Downtime) - Planned Minutes
  • Overrun is planning/time variance, not auto-wastage.
  • Job completion at 95% dispatch does not auto-convert remaining 5% to waste.
  • Remaining quantity below 100% appears as Short Close and must be operationally reviewed (carry forward, cancel balance, or dispatch later).
  • Manager/dispatch can click Close Gap with reason, and that pending short-close quantity is moved into Closed as Wastage.

9. Shift & Machine Schedule Config

  • Effective From/To: set validity date range of this shift plan.
  • Net Shift Hours: real productive hours after breaks.
  • Machine Work Schedule checkbox:
  • Checked means machine runs that day/shift.
  • Unchecked means machine is OFF that day/shift.

10. Dashboard Module (How to Read)

  • Top KPIs: OEE, Availability, Performance, Quality.
  • Planning KPIs: Planned vs Actual minutes, variance, schedule utilization.
  • Trend and analysis: production charts, machine/operator ranking, downtime, shift comparison, Pareto, at-risk jobs.
  • Short-close KPIs: pending short close and closed as wastage.

11. Search & Ease-of-use Helpers

  • Production Entry includes a user-friendly job card search picker.
  • Dispatch Entry includes job card search.
  • Machine is auto-picked from Job Card in Production Entry.

12. Archived Records

  • Archived Records lets you view deleted Job Cards, Production, and Dispatch records.
  • Use the All view to see every archived entity type together.
  • Restoring a record reactivates it while preserving audit trace.

13. Daily Working Discipline (Recommended)

  • Planner updates Job Card with correct machine before release.
  • Production logs shift-wise data same day.
  • Dispatch logs delivery same day.
  • Manager reviews short-close and overrun reasons daily.
  • Shift config is updated when timing/working pattern changes.
